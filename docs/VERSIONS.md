# Tula Versions

## Tula Version 1
The base of the language. This version is not statically typed, and is designed
to be a simple language. It is single threaded, and does not have any async-like
implementation.


## Tula Version 2
This version expands on Tula v1 by adding a async-like structures


## Tula Version 3
This version expands on Tula v2 by adding a multithreading structure


## Tula Version 4
This version expands on Tula v4 by adding some static typing in the language,
controllable by compile flags.
