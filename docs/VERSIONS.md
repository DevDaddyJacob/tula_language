# Tula Versions

## Feature Roadmap
- [v2] error handling structure (emit, attempt)
- [vTBD] async-like structures
- [vTBD] multithreading structures
- [vTBD] static typing, controllable by compile flags

## Tula Version 1
The base of the language. This version is not statically typed, and is designed
to be a simple language. It is single threaded, and does not have any async-like
implementation.


## Tula Version 2
This version expands on Tula v1 by adding error handling via the attempt library
